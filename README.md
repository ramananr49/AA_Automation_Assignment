# AutomationAnywhereTask

### This repository contains Automation Assignment of UI and API Testing using Cypress.

## Project Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ramananr49/AA_Automation_Assignment.git

2. Navigate to the project folder:
    ```bash
   cd AA_Automation_Assignment

3. Install dependencies:
    ```bash
   npm install

## Running Tests

### Open Cypress Test Runner (Interactive Mode)
    npx cypress open

### Run Cypress Tests in Headless Mode (CLI)
    npx cypress run

### Run Tests in a Specific Browser (e.g., Chrome)
    npx cypress run --browser chrome

### Run Tests in a Specific Spec File
    npx cypress run --spec "cypress/e2e/file_name.cy.js"
